# 🧪 Testing

The project includes test suites for:

- C#
- VB.NET
- C++
- NativeAOT behavior
- Memory correctness
- RNG correctness
- AES / AES-GCM / ChaCha20

## Test Categories

### 1. Next* Tests
Validate deterministic sequences.

### 2. Rng* Tests
Validate secure random generation.

### 3. Decimal Tests
Validate bit‑level correctness.

### 4. AES / GCM / ChaCha20
Validate encryption and decryption.
